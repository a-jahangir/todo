# React Todo App

A simple and responsive Todo List application built with React. This project was created as a practice to improve skills in component-based architecture, routing, and API handling.

## 🚀 Features

* Add, edit, and delete tasks
* Clean and responsive UI using Bootstrap
* Client-side routing with React Router
* Alert system using SweetAlert2
* API requests using Axios (if connected to backend)

## 🛠 Tech Stack

* React 18
* React Router DOM
* Axios
* Bootstrap 5
* SweetAlert2

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/a-jahangir/todo.git
cd todo
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

## 🧪 Available Scripts

* `npm start` – Runs the app in development mode
* `npm run build` – Builds the app for production
* `npm test` – Launches the test runner
* `npm run eject` – Ejects the app configuration
